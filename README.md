# LockTime26
The spiritual successor to [LockTimeX](https://github.com/iVintageLover/LockTimeX), skipping the line and bringing the ultra-modern and tall aesthetics of iOS 16 through iOS 26 straight to your older device running iOS 9 or later, so you can finally modernize that one iPhone 4s rotting in your drawer.

### Features
* Completely rewritten code of LockTimeX;
* Custom colors for Date and Time;
* Supported styles: iOS 26 Tall, Rail, Stencil, Rounded, Black. (more coming soon);
* Adjustable size;
* adjustable position;
* Optional 24-hour time.

### Planned features:
* Default widgets from iOS 16+;
* Depth effect for default wallpapers(?);
* More default fonts from iOS 16;
* Support for custom fonts.

### Compatibilty & Support:
The functionality of this tweak is dependent on [Matchstic's Xen HTML](https://github.com/Matchstic/Xen-HTML), making it compatible with iOS 9-14 (could potentially work on iOS 15 if you're using a rootful jailbreak, YMMV).

You can join our [Discord server](discord.gg/a7PMKvTTzB) for support if you're having any issues with this tweak.
